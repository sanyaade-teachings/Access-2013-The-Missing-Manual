Before running the TransferNames query in the Marketing.accdb database, follow these steps:
1. Open ther Marketing.accdb database and click Enable Content to enable the database.
2. Open the open the query in design view (right-click it and choose Design).
3. Click the Query Tools | Design --> Query Type --> Append button (which shows the Append dialob box).
4. Click the Browse button, and browse to (and select) the Sales.accdb database, wherever you've placed it on your computer.
